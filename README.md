# Install <br> 
cd st & make && sudo make install <br> 

# some keybinds<br> 
ctrl + shift + c for copy 
ctrl + shift + v for paste 
alt + comma for zoom in 
alt + . for zoom out 
alt + g for resetting zoom 
<br> 

# themes/fonts used 
ls-icons : https://github.com/Yash-Handa/logo-ls <br>
